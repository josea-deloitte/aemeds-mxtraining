# Quote Block

Featured quote block with a text panel (quote, author, and disclaimer) next to an image. The JavaScript is minimal: it only adds the `quote-block` class to the block; all the design lives in the CSS, which relies on the **order of the paragraphs** and the cell structure.

## 1. Authoring Contract

The block uses **one row** with **two cells**:

- **Cell 1**: the quote text. The order of the paragraphs matters because the CSS styles them by position:
  - **Paragraph 1** → large quote text (with a red divider line below it, generated by CSS).
  - **Paragraph 2** → author name.
  - **Paragraph 3** → disclaimer / secondary text.
- **Cell 2**: the image that accompanies the quote.

### Conceptual Structure

```text
| quote | |
| "Featured quote text."          | [image] |
| Author Name                     |         |
| Disclaimer or legal text.       |         |
```

(The three paragraphs all go in the first cell; the image occupies the second cell.)

## 2. CSS Customization

`quote.css` does not define its own CSS variables. It uses fixed colors and layout:

- Block background `#eff6f9`, `border-radius: 12px`, `max-width: 1200px`.
- Row layout (text `flex: 2`, image `flex: 1`) on desktop; stacks into a column below 900px.
- Quote (paragraph 1): `1.4rem`, bold, color `#1a4a5c`, with a red divider `#c0392b` (40x3px) generated via `::after`.
- Author (paragraph 2): `1rem`, 600, color `#1a4a5c`.
- Disclaimer (paragraph 3): `0.85rem`, color `#555`.
- On mobile (`width < 900px`) the card loses its radius, the text is centered, and the image extends to `100vw`.

## 3. Notes

- The styling depends on the **order** of the paragraphs (`nth-child(1/2/3)`). Authoring the paragraphs in the expected order (quote, author, disclaimer) is part of the contract.
